# pi_asap_power_agent

A power agent project with comprehensive documentation.

## Documentation

This repository contains the following documentation files:

- `documentation/agent_websockets.txt` - WebSocket agent implementation details
- `documentation/customization_clientEvents.txt` - Client event customization guide
- `documentation/customization_RAG.txt` - RAG (Retrieval-Augmented Generation) customization
- `documentation/quickstart_javascript.txt` - JavaScript quickstart guide
- `documentation/simulate_conversations.txt` - Conversation simulation documentation

## Getting Started

Refer to the documentation files in the `documentation/` directory for detailed information about using and customizing the power agent.
